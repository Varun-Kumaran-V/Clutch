Project name: Clutch

Idea: 

Input - handwritten notes, text documents in English and Indian regional languages.

Process - Extract and understand the content, identify important topics, definitions, formulas, illustrations.

Output - Generate flashcards, MCQ and other formats
Novelty-Regional languages, Reasoning with handwritten notes

Planned Features
-Mock Tests & Past Papers: Conduct quick tests, auto-gather previous exam questions.
-Analysis & Prioritization: Determine topic importance, exam relevance, learning priorities, weak-topic identification.
-Personalized Support: Personalized revision plans, adaptive learning recommendations, multilingual explanations, progress tracking.
